## Distributed File System in Rust for CCOM4017

#### Running

```./copy <PARAM1> <PARAM2>```

#### Building

If you wish to compile the code, install rust and cargo
[Link](https://www.rust-lang.org/en-US/install.html)

Then just run build

```cargo build```

If you wish to run a specific algorithm;

```cargo run --bin copy ```
